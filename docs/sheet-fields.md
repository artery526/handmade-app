# 商品試算表欄位

| 欄位 | 用途 |
| --- | --- |
| 上架 | 填 `是` 或留空代表顯示；填 `否`、`下架`、`0` 代表不顯示。 |
| 分類 | 例如 `編織花束`、`編織杯墊`、`編織髮圈`。 |
| 名稱 | 商品或作品名稱。 |
| 價格 | 商品價格，填數字即可。 |
| 庫存 | 可販售數量；`0` 會顯示為已售完。 |
| 狀態 | 例如 `現貨`、`可詢問`、`預購`、`已售完`。 |
| 圖片 | 商品圖片網址，可先留空。 |
| 描述 | 商品介紹文案。 |
| 規格 | 尺寸、入數、可客製選項等補充資訊。 |
| 排序 | 數字越小越前面。 |
